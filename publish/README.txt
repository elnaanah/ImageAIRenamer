📦 الملف التنفيذي المستقل
═══════════════════════════════════════════════════════════════

هذا ملف تنفيذي واحد مستقل (Standalone) - لا يحتاج لتثبيت أي شيء!

✅ الملف: ImageAIRenamer.exe
✅ الحجم: ~69 MB
✅ النظام: Windows 10/11 (64-bit)
✅ الحالة: جاهز للاستخدام مباشرة

═══════════════════════════════════════════════════════════════
🚀 كيفية الاستخدام
═══════════════════════════════════════════════════════════════

1. انسخ ملف ImageAIRenamer.exe إلى أي مكان تريده
2. انقر نقراً مزدوجاً لتشغيله
3. لا حاجة لتثبيت أي شيء - الملف مستقل تماماً!

═══════════════════════════════════════════════════════════════
📋 المتطلبات
═══════════════════════════════════════════════════════════════

- Windows 10 أو Windows 11 (64-bit)
- مفاتيح Google Gemini API (احصل عليها من https://aistudio.google.com/)

═══════════════════════════════════════════════════════════════
🔑 الحصول على API Key
═══════════════════════════════════════════════════════════════

1. افتح المتصفح واذهب إلى: https://aistudio.google.com/
2. سجل الدخول باستخدام حساب Google
3. اضغط على "Get API Key" أو "Create API Key"
4. انسخ المفتاح واستخدمه في التطبيق
